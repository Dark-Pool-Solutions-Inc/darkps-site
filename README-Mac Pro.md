# Dark Pool Solutions site package

## Run locally
1. Open a terminal in this folder.
2. Run `npm install`
3. Run `npm run dev`
4. Open the local URL shown by Vite.

## Deploy fast with Vercel
1. Create a new GitHub repo and upload this folder, or drag it into Vercel.
2. In Vercel, import the project.
3. Framework preset: Vite.
4. Build command: `npm run build`
5. Output directory: `dist`
6. Add your domain `darkps.com` in Vercel.
7. In GoDaddy DNS, point the domain to Vercel.

## Domain notes
- Main site should be `darkps.com`
- Redirect `dpsix.com` to `darkps.com`
- Email stays `@darkps.com`
